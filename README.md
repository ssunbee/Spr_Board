# 📋Spring게시판만들기
가장 기본적이고 보편적인 게시판 기능을 둘러볼 수 있는 서비스입니다. 스프링 관련 기술들 , 자바 11 기능들, 개발 도구들을 경험할 수 있도록 만들어졌습니다.
서울우유 웹사이트를 리뉴얼해서 게시판과 회원관리, 로그인, 회원가입 기능을 추가하였습니다.
